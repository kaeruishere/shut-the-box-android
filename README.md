# Shut The Box - Mobile

[![Android](https://img.shields.io/badge/Android-3DDC84?style=for-the-badge&logo=android&logoColor=white)](https://developer.android.com)
[![Java](https://img.shields.io/badge/Java-ED8B00?style=for-the-badge&logo=openjdk&logoColor=white)](https://www.java.com)

Klasik masa oyunu **Shut The Box**'ın Android cihazlar için Java ile yazılmış yerel (native) mobil sürümüdür.

## 🎲 Oyun Kuralları

1. Oyuncu iki adet zar atar.
2. Atılan zarların toplamına veya zarların kendi değerlerine karşılık gelen sayı plakalarını (1-9 arası) kapatır.
3. Kapatacak plaka kalmayana kadar zar atmaya devam eder. Amaç tüm plakaları kapatıp "Shut The Box" yapmaktır.

## 📱 Özellikler

- **Zar Animasyonları:** Gerçekçi zar yuvarlanma animasyonları ve ses efektleri.
- **Skor Takibi:** En düşük ceza puanlarını ve kazanma serilerini tutan yerel skor tablosu.
- **Kullanıcı Dostu Arayüz:** Tek elle rahatça oynanabilecek dikey tasarım.

## 🛠️ Geliştirme Notları

- **IDE:** Android Studio
- **Dil:** Java (Native Android SDK)
- **Minimum SDK:** API 21+
